## ðŸ“· How to Add Screenshots

### For Contributors

1. Take screenshots on your device:
   - **iOS Simulator**: `Cmd + S` or Screenshot tool
   - **Android Emulator**: Screenshot button in toolbar
   - **Physical Device**: Use device's screenshot function

2. Optimize images:
   - Resize to reasonable dimensions (e.g., 1080px width)
   - Use PNG or JPEG format
   - Keep file size under 500KB per image

3. Upload to repository:
   ```bash
   # Create screenshots folder if it doesn't exist
   mkdir -p assets/screenshots
   
   # Add your screenshots
   git add assets/screenshots/
   git commit -m "docs: add app screenshots"
   ```

4. Update this file with markdown:
   ```markdown
   ![Home Screen](assets/screenshots/home-screen.png)
   ```
